## 数组中字符串的最大值

[地址](https://leetcode.cn/problems/maximum-value-of-a-string-in-an-array/)

```ts
function maximumValue(strs: string[]): number {
    // 使用正则判断是否存在字符串
    const reg = /[a-z]/
    let num = 0
    for (const str of strs) {
        num = Math.max(num, reg.test(str) ? str.length : Number.parseInt(str))
    }

    return num
};
```
