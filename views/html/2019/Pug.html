    #pug 简介

    pug 其实就是一种模板引擎。最终它将渲染成 html 的格式，只不过它比 html 简洁，所以有时候会用 pug 来学 html 代码。近期希望自己总结一下语法，可以时常回顾，而不需要翻看各类介绍。
