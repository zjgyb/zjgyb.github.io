    es6学习

    Tony9/16/2018es6js

    已经使用了一段时间 es6 了，感觉应该对其中的知识点进行总结，能够时常回顾，加深记忆。

    #let&const

    let的主要作用是能够存在作用域，这一点是有别于var的，举个例子

    var a = [];
for (var i = 0; i < 3; i++) {
  a[i] = function() {
    return i;
  };
}

a[0](); // 3
a[1](); // 3
a[2](); // 3

    let a = [];
for (let i = 0; i < 3; i++) {
  a[i] = function() {
    return i;
  };
}

a[0](); // 0
a[1](); // 1
a[2](); // 2

    #箭头函数

    在 es6 中我最偏爱的就是箭头函数了，它的简洁是我偏爱的最大原因，举个例子

    // es5
var a = function() {
  return 1;
};

// es6
const a = () => 1;

    #构造函数

    // es5
var Person = function(name) {
  this.name = name;
};
var one = new Person("nicegong");

// es6
class Person {
  construct(name) {
    this.name = name;
  }
}
const one = new Person("nicegong");

    #export&import

    export是从文件中导出变量与函数，import是从文件中引用变量与函数

    // add.js
const add = (a, b) => {
  return a + b;
};

export { add };
