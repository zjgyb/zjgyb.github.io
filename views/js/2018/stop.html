    stopPropagation和preventDefault的区别

    Tony11/26/2018stopPropagationpreventDefaultjs

    {% note primary %}
    之前看过stopPropagation和preventDefault的区别，然而当我再次遇到的时候已经忘了两者的区别，因此我需要写一篇关于两者区别的文章，以便理解以及之后忘记时能够通过阅读这篇文章回忆起来。
    {% endnote %}

    #stopPropagation

    阻止捕获和冒泡阶段中当前事件的进一步传播。

    #preventDefault

    阻止浏览器做出默认操作。
